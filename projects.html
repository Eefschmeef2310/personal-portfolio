Games

Games that I have contributed to, that can all be played through my Itch.io page.

Arcane Raiders

Arcane Raiders is an online/local multiplayer game that has been published on Steam. For this project, I designed and implemented all spells, as well as implementing shaders and additional VFX.

Arcane Raiders was displayed at the UTS Tech Fest 2024 Student Showcase, where it won first place for best game in 
Game Design Studio 1.

Additionally, Arcane Raiders was showcased at the SXSW Sydney 2024 Main Games Showcase, where it was shortlisted for Best Student Game.

Learn More

Pascal's Requiem

As a part of the UTS Playmaker's Development Team 2024, Pascal's Requiem is a 2.5D RPG created in the Unity Game Engine.

For this project, I acted as a technical artist, using particle effects and shaders to bring the visuals of the game to life.

Learn More

Smoothie Maker Pro

A frantic puzzle game created in 48 hours for the Pixel Pasture Invitational Game Jam 2024. I contributed with ensuring positive user experience.

The game achieved 3rd overall place, with the highest public voting.

Nothing Shady at THIS BBQ!

This action game achieved 2nd place at the four-day UTS Playmakers Summertime Game Jam 2024. I contributed through creating the tutorial, food system, and menus.

Starship Scramble

Created by myself and other members of Team Stingray for the Game Design Methodologies subject, where it was presented at the UTS Student Games Showcase 2023.

Super Boxing

Created in a single fortnight, Super Boxing was a recreation of the original Boxing game for the Atari 2600, created in the Godot engine.

For this project, I worked on the shader materials and particles effects.

Additional Projects

In addition to published games, I have experimented with creating software for different platforms.

Godot - Shader Repository

A shader repository build entirely in Godot, storing shaders used for other games, shaders translated from Shadertoy, and recreations of shaders from other game engines.

Group Project - GeneRACE!

This projected demonstrated Unreal Engine knowledge taught in Advanced Games Programming, including C++ programming and multiplayer implementation.

For this project, I implemented the procedurally generated race track and the visual aspects of the game.

Group Project - Doodle Jump

Created in three.js, this recreation of the popular mobile game Doodle Jump utilizes JavaScript and WebGL to make a game playable entirely within a web browser.

For this project, I implemented the core visuals and shader effects.

Fragment Shader TV

This project demonstrates model loading and custom fragment shaders effects in three.js. It received full marks.

You can view the website it is hosted on, or view the provided video!
